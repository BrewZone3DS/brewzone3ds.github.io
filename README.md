# BrewZone Web

The official website for BrewZone. This site serves as the primary hub for installation, documentation, and news.

---

## Web Features

The site is in very early stages of development, so right now it only contains the following:

* **Installing:** Download the latest builds or updates for your console (when the app releases).
* **Documentation:** Detailed guides for setting up BrewZone.

More is planned for the website in the future, so stay tuned!
